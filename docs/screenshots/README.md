# Screenshots

Drop PNG screenshots here with these filenames to populate the main README:

| Filename | Shows |
| --- | --- |
| `hero.png` | The top of the manual — title, search bar, first visible items |
| `banners.png` | The green "What's New" banner + blue "Recent User Additions" banner |
| `modal.png` | A clicked-open item modal showing the explainer text |

## How to capture them (macOS)

1. Open the manual: `open ~/.hermes/docs/Danual.html`
2. Maximize the browser window (cleaner shots).
3. Press **Cmd + Shift + 4**, then **Space**, then click the browser window — captures just the window with a shadow.
4. For region shots (like a single banner), use **Cmd + Shift + 4** and drag the selection.
5. Save the resulting PNG from your Desktop into this directory with the filename above.

## How to capture them (Linux)

Use `gnome-screenshot -w` for window, `gnome-screenshot -a` for area, or any tool like Flameshot / Spectacle. Save with the filenames above.

## Tips

- A dark OS theme matches Danual's dark UI best.
- Keep each image under ~400 KB; PNG is fine at this size.
- If you're submitting a PR with code changes, screenshots aren't required — these are just for the README showcase.
